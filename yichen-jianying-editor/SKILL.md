# 剪映剪辑助手

直接操作剪映 UI 时，使用 Computer Use。点击、拖拽、输入之前，先获取 `com.lemon.lvpro` 的应用状态。

## 核心流程

1. 先确认素材路径和文件信息。
   - 检查文件是否存在、大小是否正常。
   - 需要确认时长、分辨率、帧率时，用 `ffprobe`。
2. 打开剪映，创建新项目或继续当前项目。
3. 导入素材。
   - 优先使用剪映的导入按钮。
   - 如果 macOS 文件选择器不容易输入中文路径或中文文件名，就在项目工作区复制一个英文名工作副本，再导入工作副本。
   - 除非用户明确要求，不要改动原始素材。
4. 把素材放入时间线。
   - 优先点击媒体缩略图右下角/角落附近的“添加到轨道”区域。
   - 如果点击缩略图主体只是在预览，不要反复硬拖主体。
   - 如果误触发播放，先暂停，再点缩略图角落/覆盖按钮。
   - 成功后确认主时间线出现片段，并且右侧面板切到视频参数控制区。
5. 剪映内剪辑或精修前，尽量先确认用户要处理的是原片、已粗剪 MP4，还是当前时间线素材。
6. 如果用户要“自动粗剪口播”“基于转写删重复/停顿/断句”，不要在本 skill 内处理；应使用火山 ASR/自动剪辑 skill 生成通过审核的粗剪 MP4，再回到剪映导入精修。
7. 剪映内主要负责：
   - 导入原片或已粗剪 MP4。
   - 放入主时间线。
   - 调整画面、裁切、缩放、位置、封面帧。
   - 添加或校正字幕、标题、贴纸、音效、音乐。
   - 做用户指定的手动剪切、微调和导出。
8. 用户要求边做边记录时，每完成一个关键动作就更新项目日志。

## 剪映 UI 经验

- 应用 bundle id：`com.lemon.lvpro`。
- 媒体区会显示已导入视频缩略图和时长。
- 点击缩略图主体可能只是预览素材。
- 时间线在窗口下半部分。素材成功加入后，会出现带缩略图的片段条。
- 右侧出现 `画面 / 音频 / 变速 / 动画 / 调整 / AI效果` 时，通常说明时间线视频片段已被选中。

## 和自动粗剪的边界

这个 skill 只记录剪映内的操作。不在剪映里面发生的步骤，例如火山 ASR 转写、根据时间戳生成 `delete_segments.json`、文件级自动粗剪、三层 ASR 审核，都不属于本 skill。

如果用户给的是已经通过自动粗剪审核的 MP4，本 skill 的职责是：

1. 把粗剪 MP4 导入剪映。
2. 放入时间线。
3. 做视觉和听感精修。
4. 导入或校正字幕。
5. 按用户要求导出。

## 日志格式

记录项目时，在日志末尾追加简洁条目：

```markdown
## YYYY-MM-DD HH:MM

- Action: 导入素材 / 加入时间线 / 剪映内精修 / 字幕 / 导出。
- Details: 文件名、时间码、时长、界面结果。
- Decision: 为什么这样处理。
- Next: 下一步做什么。
```

日志要实用，重点是方便恢复项目和复盘剪辑决策，不要写成流水账。
